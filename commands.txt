모든 명령어의 시작은 /i 로 통일합니다.
/i command : 명령어 목록이 있는 사이트의 링크를 제공합니다.
/i info : 이 봇의 정보를 전송합니다.
/i made by : 개발자의 깃허브 닉네임과 링크를 전송합니다.
/i search hp '멤버이름' : 멤버에 대한 정보를 전송합니다. 
/i link : 해피플레이스 서버의 초대 링크를 전송합니다.
/i rules : 해피플레이스 프라이빗 서버의 규칙을 전송합니다. 봇과의 DM에서만 사용할 수 있습니다.
/i g_drive : 얼음봇의 소스 코드가 담겨 있는 구글 드라이브 페이지로 이동합니다. 봇 개발자만 사용할 수 있습니다.
/i github : 얼음봇의 소스 코드가 담겨 있는 깃허브 페이지로 이동할 수 있는 링크를 전송합니다.
/i blacklist : 해피플레이스의 블랙리스트를 전송합니다.
/i update : 봇이 업데이트 되도록 달성을 소환합니다.

- 5번째 줄의 '멤버이름' 은 다음과 같습니다. {달성, 삐약, 관제, 러디, 초바, 핀오, 얼음, 낭자, 리엔, 원상, 후디, 오버, 공주, 하보리, 설빙, 심심, 규진}
